# ⚽ Football Data Analysis Project

## 📌 Overview

This project focuses on analyzing football player data to help a club recover from financial difficulties by increasing the overall **market value of players by 15%** in the upcoming season.

The analysis simulates the role of a **Data Analyst at a football club**, where data-driven insights are used to improve player performance and valuation.

## 🎯 Problem Statement

The football club is facing financial challenges and aims to increase player market value.
This project analyzes key factors such as **performance metrics, age, position, and league impact** to identify strategies for value growth.

## 📊 Objectives

* Identify players with the highest market value
* Analyze factors affecting player valuation
* Compare market value across leagues and clubs
* Understand performance vs valuation relationship
* Provide recommendations to improve player worth

## 🗂️ Dataset

The project uses three datasets:

* **players.csv** → Player details and market value
* **appearances.csv** → Match performance data
* **games.csv** → Match and season information
  
# ⚙️ Data Processing Steps

1. Data Merging (players + appearances + games)
2. Data Cleaning (handling missing values & duplicates)
3. Feature Selection (removing unnecessary columns)
4. Sorting & structuring for analysis


# 🔍 Key Analysis Performed

* Top players by market value
* Impact of goals, assists, and minutes played
* League-wise and club-wise market comparison
* Factors influencing player valuation
* Performance vs market value relationship


# 📈 Insights

* Player performance (goals & assists) strongly impacts market value
* Certain positions (like forwards) tend to have higher valuations
* Younger players often show higher growth potential
* League visibility influences player pricing
* Some players have higher value due to external factors beyond performance


# 🧠 Conclusion

The analysis highlights that improving **performance metrics, player development, and strategic positioning** can significantly boost player market value and help the club achieve financial recovery.
